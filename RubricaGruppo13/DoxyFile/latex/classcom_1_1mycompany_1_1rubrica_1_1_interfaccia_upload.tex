\doxysubsection{Detailed Description}
Classe che rappresenta l\textquotesingle{}interfaccia per caricare e gestire file di contatti. 

Tramite questa interfaccia, l\textquotesingle{}utente potrà selezionare o trascinare un file da caricare nella rubrica.

\begin{DoxyNote}{Note}
Supporta solo file con estensione {\ttfamily .txt}.
\end{DoxyNote}
