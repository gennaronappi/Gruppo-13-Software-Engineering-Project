Detailed Description

Classe che rappresenta l'interfaccia per caricare e gestire file di contatti.

Tramite questa interfaccia, l'utente potrà selezionare o trascinare un file da caricare nella rubrica.

Note
Supporta solo file con estensione .txt.
Author
Francesco, Antonio, Gennaro
